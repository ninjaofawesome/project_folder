CSS Selector Statements

HTML uses tags like <b> and <i> for formatting - like bold or italic text.These HTML tags are called formatting tags.

<b> defines bold text

<m> defines emphasized text

<i> Defines a part of text in an alternate voice or mood

<small> Defines smaller text

<strong> Defines stronger text

<sub> Defines subscript text

<sup> Defines superscript text

<ins> Defines inserted text

<del> Defines deleted text

For more Formatting tags visit W3 Schools HTML Text Formatting Section
