Tables and Forms

A table adds credibility to your site. Very respectable. Put tables everywhere.

Things That Are Sparrows Things That Are Not Sparrows
That sparrow Dogs
That other sparrow Styrofoam
The sparrow over there Napkins

You can create a respectable table by manually building the rows and cells, using the following tags:
  • <table></table> to start/end the table
  • ltth></th> for table headers
  • <tr></tr> for table rows
  • <td></td> for table cells

< table border="1" >
< tr>
<th>Things That Are Sparrows</th>
<th>Things That Are Not Sparrows</th>
</tr>
<tr>
<td>That sparrow</td>
<td>Dogs</td>
*lt/tr>
<tr>
<td>That other sparrow</td>
<td>Styrofoam</td>
</tr>
<tr>
<td>The sparrow over there</td>
<td>Napkins</td>
</table>
